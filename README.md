# Socket Client-Server System (TCP/UDP)

## Описание

Данный проект представляет собой реализацию клиент-серверной архитектуры с использованием сетевых сокетов на языках C/C++.

Поддерживаются два протокола:

* **TCP** — надёжная передача данных с подтверждением
* **UDP** — быстрая передача с собственной логикой подтверждений и повторов

Проект был выполнен в рамках лабораторной работы по дисциплине *«Операционные системы»*.

---

## Функциональность

### TCP

* Подключение клиента к серверу
* Команды:

  * `put` — отправка сообщений на сервер
  * `get` — получение сохранённых сообщений
* Подтверждение каждого сообщения (`ok`)
* Сохранение сообщений в `msg.txt`
* Поддержка нескольких клиентов (через `select`)

### UDP

* Отправка сообщений без установления соединения
* Повторная отправка неподтверждённых сообщений
* Дедупликация сообщений на сервере
* Поддержка нескольких портов
* Автоочистка неактивных клиентов

## Структура проекта

```
.
├── tcpclient.cpp     # TCP клиент (put)
├── tcpclient2.cpp    # TCP клиент (get)
├── tcpserver.c       # TCP сервер
├── udpclient.c       # UDP клиент
├── udpserver.c       # UDP сервер
├── msg.txt           # Файл с сохранёнными сообщениями
└── input.txt         # Входной файл с сообщениями
```

## Сборка

### Linux (gcc / g++)

```bash
gcc tcpserver.c -o tcpserver
gcc udpclient.c -o udpclient
gcc udpserver.c -o udpserver

g++ tcpclient.cpp -o tcpclient
g++ tcpclient2.cpp -o tcpclient2
```

### Windows (MinGW / MSVC)

Для WinSock необходимо подключение библиотеки:

```bash
g++ tcpclient.cpp -lws2_32 -o tcpclient.exe
g++ tcpclient2.cpp -lws2_32 -o tcpclient2.exe
```

##  Запуск

### TCP сервер

```bash
./tcpserver 12345
```

### TCP клиент (отправка)

```bash
./tcpclient 127.0.0.1:12345 input.txt
```

### TCP клиент (получение)

```bash
./tcpclient2 127.0.0.1:12345 get output.txt
```

---

### UDP сервер

```bash
./udpserver 12345 12350
```

### UDP клиент

```bash
./udpclient 127.0.0.1:12345 input.txt
```

## Формат входных данных

Каждая строка файла:

```
HH:MM:SS HH:MM:SS BBB Message text
```

Пример:

```
12:00:01 12:00:02 123 Hello world
```

## Тестирование

### TCP

1. Запустить сервер
2. Запустить клиент `put`
3. Проверить `msg.txt`
4. Запустить клиент `get`
5. Убедиться, что данные корректно получены

### UDP

1. Запустить UDP сервер
2. Запустить UDP клиент
3. Проверить:

   * повторную отправку
   * отсутствие дубликатов
   * корректность `msg.txt`

## Особенности реализации

* Неблокирующие сокеты
* Использование `select` для мультиплексирования
* Гарантированная отправка данных (`send_all`)
* Дедупликация UDP сообщений через hash set
* Поддержка частичных передач и восстановления
